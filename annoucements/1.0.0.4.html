WSBSD 1.0.0.4 Announcement

We, TTS Consulting, are pleased to announce WSBSD 1.0.0.4, focused on adding login screens to supported BSD distros and more enhancements to improve the user experience.

Changelog for WSBSD 1.0.0.4

  • New Feature: Added login screens for all supported BSD distributions excluding OpenBSD and FreeBSD
  • Enhancement: Improved DragonFlyBSD 6.4.2 support

What's New in This Release

This release focuses primarily on enhancing the user authentication experience across all supported BSD distributions. Users will now encounter proper login screens that simulate the authentic BSD login experience.

System Requirements

  • Windows 7 SP1 or later
  • How Much RAM The Distro you choose Needs.
  • How Much The Distro you choose's Storage Is.
  • .NET Framework 4.7.2 or later
